Sierra

Sierra is a micro-templating engine for web frameworks in Python.

About

Sierra is a Python micro templating engine for web frameworks that makes integration faster. You can now develop your web applications purely in Python, taking advantage of its powerful functionalities with short and simple enough syntax. It can also be used standalone. You can use Python loops, functions, variables, conditions - anything really you can do on Python, with Sierra. Harness the power of Python for you web applications now!

Example

Using this with Flask makes life easier if you're developing web applications with just HTML and CSS.

JS not supported

Adding JS is not yet supported by Sierra and has to be done manually (if desired).
