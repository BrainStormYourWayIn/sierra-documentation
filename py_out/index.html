Working with Python Outputs

Displaying Python outputs on your web applications are actually pretty simple with Sierra. To work with this, you can use 'writeHTML()' , which has been covered on the main page of the documentation. Using f-strings is an easier way to print variables onto your work. writeHTML(f'''{var}''') or a.start_p(f'''{var}''') , both work.

Say you've used getTable() to load in a .csv file with pandas and write some quick code for displaying each row as a list: 

  
        this      is        a          csv
    0   word      word1     word2      word3
    1   test1     test2     test3      test4
    2   foo1      foo2      foo3       foo4

    for i in range(0, 3):
    for g in range(1, 4):
        if i+1==g:
            a = f'''r{g} = {list(df.iloc[i])}{b}'''            # var 'b' stands for <br> See 'Other Functions'
            writeHTML(a)
            # or if you want it on a paragraph
            a = tTags(True)
            a.start_p(a)

            Output:
            # r1 = ['word', 'word1', 'word2', 'word3']
            # r2 = ['test1', 'test2', 'test3', 'test4']
            # r3 = ['foo1', 'foo2', 'foo3', 'foo4']

The output is now displayed on the webpage! Of course, you can style it the way you want depending on the tag you enclosed it in.
